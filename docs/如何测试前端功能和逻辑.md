# 如何测试前端功能和逻辑

本文档介绍 DeepResearchWeb 前端的各种测试方法。

## 快速验证 (HTTP 请求)

适用于快速检查页面是否可访问，无需安装额外工具：

```bash
# 测试前端所有页面
node test-pages.js
```

### 手动测试命令

```bash
# 前端页面测试
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/login
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/register
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/chat
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/config/tools
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/config/skills
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000/config/mcp

# 后端 API 测试
curl -s http://localhost:8000/
curl -s http://localhost:8000/api/v1/health
```

## 完整浏览器测试 (Playwright)

适用于需要测试 JavaScript 交互、表单提交、页面跳转等复杂逻辑。

### 1. 安装依赖

```bash
cd frontend
npm install -D @playwright/test
npx playwright install chromium
```

### 2. 运行测试

```bash
# 运行所有测试
npx playwright test

# 运行特定测试文件
npx playwright test tests/e2e.spec.ts

# 在浏览器中打开测试
npx playwright test --ui
```

### 3. 测试配置

配置文件位于 `frontend/playwright.config.ts`：

```typescript
import { defineConfig } from '@playwright/test';

export default defineConfig({
  testDir: './tests',
  timeout: 30000,
  use: {
    baseURL: 'http://localhost:3000',
    headless: true,
    viewport: { width: 1280, height: 720 },
  },
  projects: [
    {
      name: 'chromium',
      use: { browserName: 'chromium' },
    },
  ],
});
```

测试文件：`frontend/tests/e2e.spec.ts`

| 进入聊天页面 | 测试聊天页面访问 |
| 聊天页面元素 | 验证聊天页面 UI 元素 |

## 常见问题

### Playwright 安装失败

如果 `npx playwright install chromium` 失败，尝试：

```bash
# 使用系统浏览器
export PLAYWRIGHT_CHROMIUM_EXECUTABLE_PATH=/path/to/chromium

# 或使用 Homebrew 安装
brew install chromium
```

### 测试超时

增加超时时间：

```typescript
export default defineConfig({
  timeout: 60000, // 60秒
});
```

### 调试模式

```bash
# 打开 Playwright UI
npx playwright test --ui

# 显示浏览器控制台
npx playwright test -- headed

# 逐步执行
npx playwright test --debug
```
